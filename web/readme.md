## Setup

Need this `.env`:

```
CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
```

`CLERK_` come from [Clerk](https://clerk.com) workspace.

```
bun i
```

## Run

```
bun dev
```

## Stack

- [TanStack Start](https://tanstack.com/start/latest) - react framework (routing, server actions, ..)
- [Clerk](https://clerk.com) - auth
